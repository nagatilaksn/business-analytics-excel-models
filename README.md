# Business Analytics Excel Models

This repository contains practical Excel models developed as part of MBA coursework and personal learning in business analytics and financial analysis.

The goal of these models is to apply quantitative methods to support business decision making.

## Projects Included

### 1. Monte Carlo Simulation for Profit Estimation
A simulation model that estimates profit under uncertain demand, price, and cost conditions using probability distributions.

Techniques used:
- Normal distribution for demand
- Triangular distribution for selling price
- Uniform distribution for variable cost
- 1000 simulation trials
- Expected profit and risk analysis

### 2. Optimization Model using Excel Solver
A linear optimization model developed using Excel Solver to determine optimal business decisions under resource constraints.

Applications:
- Production planning
- Profit maximization
- Resource allocation

### 3. Business Data Analysis using Excel
Business dataset analysis using Excel tools such as:
- Pivot tables
- VLOOKUP
- Data summarization

## Tools Used
- Microsoft Excel
- Monte Carlo Simulation
- Excel Solver
- Data Analysis techniques
